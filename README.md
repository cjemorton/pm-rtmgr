RTmgr

rTorrent Manager.

---

Rtmgr-Gen-Db

This module connects to a rTorrent/ruTorrent instance and returns a list of torrents storing them to a local database.


INSTALLATION

To install this module, run the following commands:

	perl Makefile.PL
	make
	make test
	make install
